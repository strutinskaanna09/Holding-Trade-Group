# 🚀 Інструкція по запуску серверів

## ⚠️ ВАЖЛИВО: Обидва сервери мають працювати одночасно!

### 📡 Крок 1: Запустіть API сервер

Відкрийте **Термінал 1** і виконайте:

```bash
cd /Users/moutamanalyasiri/Desktop/Group/apps/api
npm run dev
```

**Очікуваний вивід:**
```
🚀 API сервер запущено на http://0.0.0.0:3000
📡 Доступний на http://localhost:3000 та http://127.0.0.1:3000
```

**Зачекайте 10-15 секунд** для компіляції TypeScript.

---

### 🌐 Крок 2: Запустіть Web сервер

Відкрийте **Термінал 2** (новий термінал) і виконайте:

```bash
cd /Users/moutamanalyasiri/Desktop/Group/apps/web
PORT=3001 npm run dev
```

**Очікуваний вивід:**
```
▲ Next.js 16.0.10
- Local:        http://localhost:3001
```

**Зачекайте 10-15 секунд** для компіляції Next.js.

---

### ✅ Крок 3: Перевірка

1. **Перевірте API:**
   ```bash
   curl http://localhost:3000/categories
   ```
   Має повернути JSON з категоріями.

2. **Відкрийте браузер:**
   ```
   http://localhost:3001
   ```

---

### 🔧 Якщо помилки "fetch failed":

1. ✅ Переконайтеся, що API сервер запущений (Термінал 1)
2. ✅ Переконайтеся, що Web сервер запущений (Термінал 2)
3. ✅ Зачекайте 10-15 секунд після запуску
4. ✅ Перевірте, чи порти не зайняті:
   ```bash
   lsof -i :3000
   lsof -i :3001
   ```

---

### 🛑 Зупинка серверів

Натисніть `Ctrl+C` в обох терміналах.

---

### 📝 Порти

- **API сервер:** http://localhost:3000
- **Web сайт:** http://localhost:3001
